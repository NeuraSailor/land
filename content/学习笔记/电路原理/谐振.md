## 谐振

**电路中感性元件和容性元件的电抗效应相互抵消，使得电路呈现纯电阻性的状态。**
**统一特征**：电路的总**电抗为零 (X = 0)** 或总**电纳为零 (B = 0)**。

| 特征方面       | 统一的核心原理                                                                       | 串联谐振的表现                          | 并联谐振的表现                         |
| ---------- | ----------------------------------------------------------------------------- | -------------------------------- | ------------------------------- |
| **阻抗/导纳**  | 总电抗/电纳为零                                                                      | **阻抗最小** (Z₀=R)                  | **阻抗最大** (Z₀=Rp，Rp 为等效并联电阻)     |
| **电压电流相位** | 同相位，纯电阻性                                                                      | Vṡ 与 Iṡ 同相                     | Vṡ 与 Iṡ 同相                    |
| **频率响应**   | 在 f₀ 处响应出现峰值                                                                  | **回路电流 I** 最大                    | **端电压 V** 最大                    |
| **过激现象**   | 元件上的响应被放大 Q 倍                                                                   | **电感/电容电压** 是电源电压的 **Q 倍** (电压谐振) | **电感/电容电流** 是总电流的 **Q 倍** (电流谐振) |
| **功率特征**   | - **视在功率 (S)** 等于**有功功率 (P)**，即 S=PS=P。<br>- **无功功率 (Q)** 为零，即 Q=0 VARQ=0VAR。 |                                  |                                 |
| **电路处理**   |                                                                               | 短路（阻抗为 0）                         | 断路 （导纳为 0）                      |

## 串联谐振

1. 呈纯电阻特性
	串联的电感电容对两端等电势
	阻抗最小值，$I,U_{R}取最大值$（应为最大值，因为阻抗最小，电流最大，$U_R=IR$ 也最大），电路消耗功率最大，合无功功率为 0，与电源没有无功功率交换
2. 此时 $\boxed{w_0=\frac{1}{\sqrt{ LC }}}$（$w_0L=\frac{1}{w_0C}$）
3. 特征阻抗 $\rho=w_0L=\frac{1}{w_0C}=\sqrt{ \frac{L}{C} }$

## 并联谐振

![[assets/ccf8e4eb84737d3228250f869eca2b3d_MD5.png|300]]

1. 等效导纳呈纯电导特性
	并联的电感电容上的电流大小相等方向相反，与外电流没有电流交换，断路
	导纳最小值， $I一定时，U,I_{R}max；U一定时，I min$ ($I=U|Y|$)
2. $\boxed{w_0=\frac{1}{\sqrt{ LC }}}$
3. $Q=\frac{w_{0}c}{G}\left( \frac{容纳或导纳}{电导} \right)=\frac{Uw_{0}C}{IG}\left( \frac{电感分流}{电阻分流} \right)=\frac{U^2w_{0}C}{U^2G}  (\frac{{电感（电容）单独无功功率（两者间交换）}}{电阻有功功率})$
